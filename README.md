#  Human Detection using OpenCV

##  Project Overview
This project detects whether an image or video contains a real human using OpenCV and Haar Cascade Classifier. Users can capture an image using a live camera, upload an image, or upload a video for human detection.

##  Technologies Used
- Python
- OpenCV
- NumPy
- Matplotlib
- Google Colab
- Haar Cascade Classifier

##  Features
- Live Camera Human Detection
-  Image Upload Detection
-  Video Upload Detection
- Face Detection using Haar Cascade
- Real-time Human Detection
- Simple Interactive Menu
